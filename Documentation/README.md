# Folder structure

The project has quite simple folder sctructure

- Hynek_Janac (root folder used for storing files which are accesed the most often index.html ...)
  
  - CSS (Here are stored all the stylsheets)
    - Universal.css stylesheet used all over the site
    - Root.css light/dark mode colors panel
    - Article.css 
    - Frontpage.css 
    - Menu.css
    - Articlewriter.css
  
  - Img (images are stored in an webp or svg format)
    
    - Gallery (Images displayed in gallery)
    
    - Svg (Vector graphics)
    
    - Writing (Images used in articles)
  
  - Js (Here is stored all the javascript)
  
  - Pages (Here are stored all the main pages (Projects, resources, writing, donate, gallery))
  
  - Writing (This a place where all the articles live)
